RFID Door Lock

Java

A sensor recording system was created to record status of a door lock. A database was set up on a server to record the RFID tag, date and time and success/fail of the attempt. Authorized tag ID's were stored in the database with corresponding doors. So that only the correct door is opened when a tag is scanned. The lock and motor were communicated with using MQTT.
Data was sent in JSON format.

An Android Application was also created to open the lock.

An overview of the process:

  • Create RFID object from scanned tag
  • Convert to JSON string and send to server
  • A SQL select statment is used to select all valid tags in database which match scanned tag
  • If true then another check is done on the door database to see if this tag matches this door
  • This attempt will then be inserted into the database
  • A valid or invalid JSON string will be returned, then a MQTT will start to broadcast to the lock telling it to open if valid
