Charging Points

Java - Android Studio

Android application to aid users of electric vehicles to easily find a suitible charger for their vehicle based on location. Then directions will be offered through Google Maps. Only the required data will be gathered through parameters in the GET API request, maintaining a lightweight fast app for use on the move.

This project uses:

  • Deserialization of JSON string to objects
  • REST API
  • Database and SQL
  • Google Maps API integration
  • Designing in Android Studio and Google Maps skins

Node.js - React Native

Application very similar to Twitter. There were multiple API end points to reach, some of which required the user to be logged in and a authorisation to be passed with the API. This application makes use of GET, POST, UPDATE and DELETE requests.

Some of these are:

  • Create Account
  • Log In
  • Post tweet
  • Follow / Unfollow
